quatnormalize

Inverse of a quaternion.

Syntax

q = quatnormalize(q)

Description

q = quatnormalize(q) normalizes the specified quaternion.

Inputs

Variable Symbol Description Format Units
q quaternion 4×1
double		 -

Outputs

Variable Symbol Description Format Units
q unit quaternion 4×1
double		 -

Example

Normalize the qauternion

Defining the quaternion,

q = [1;2;3;4];

Normalizing the quaternion,

q = quatnormalize(q)

q =

    0.1826
    0.3651
    0.5477
    0.7303

Verifying that the updated quaternion has magnitude 1,

q_norm = quatnorm(q)

q_norm =

    1.0000
