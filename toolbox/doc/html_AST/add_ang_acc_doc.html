Syntax

alpha_A2C_C = add_ang_acc(omega_A2B_B,alpha_A2B_B,omega_B2C_C,alpha_B2C_C,R_B2C)

Inputs

Variable Symbol Description Format Units
omega_A2B_B angular velocity of frame with respect to frame , expressed in frame 3×1
double		 rad/s
alpha_A2B_B angular acceleration of frame with respect to frame , expressed in frame 3×1
double		 rad/s²
omega_B2C_C angular velocity of frame with respect to frame , expressed in frame 3×1
double		 rad/s
alpha_B2C_C angular acceleration of frame with respect to frame , expressed in frame 3×1
double		 rad/s²
R_B2C passive rotation matrix from frame to frame 3×3
double		 -

Outputs

Variable Symbol Description Format Units
alpha_A2C_C angular acceleration of frame with respect to frame , expressed in frame 3×1
double		 rad/s²
