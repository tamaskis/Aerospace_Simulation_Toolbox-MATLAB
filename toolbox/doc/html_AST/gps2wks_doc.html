gps2wks

GPS week and seconds from GPS time.

Syntax

[wk,s] = gps2wks(MJD_GPS)

Description

[wk,s] = gps2wks(MJD_GPS) determines the GPS week and seconds given the GPS time.

Inputs

Variable Symbol Description Format Units
MJD_GPS GPS time 1×1
double		 MJD

Outputs

Variable Symbol Description Format Units
wk GPS week 1×1
double		 wk
s GPS seconds 1×1
double		 s

Example

Find the GPS week and seconds corresponding to February 1, 2022, 21:35:27 GPS.

First, let's find the modified Julian date of GPS time.

MJD_GPS = cal2mjd([2022,2,1,21,35,27]);

Finding the GPS week and seconds,

[wk,s] = gps2wks(MJD_GPS)

wk =

        2195


s =

      250527
