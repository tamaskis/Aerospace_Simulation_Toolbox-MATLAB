Syntax

theta_rad = arcsec2rad(theta_arcsec)

Description

theta_rad = arcsec2rad(theta_arcsec) converts an angle from arcseconds () to radians ().

Inputs

Variable Symbol Description Format Units
theta_arcsec angle in arcseconds 1×1
double		 ''

Outputs

Variable Symbol Description Format Units
theta_rad angle in radians 1×1
double		 rad

Example

Convert to radians.

theta_arcsec = 1;
theta_rad = arcsec2rad(theta_arcsec)

theta_rad =

   4.8481e-06
