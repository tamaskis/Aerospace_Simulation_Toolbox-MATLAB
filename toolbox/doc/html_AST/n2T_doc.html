n2T

Orbital period from mean motion.

Back to Orbital Mechanics.

Contents

  • Syntax
  • Inputs
  • Outputs
  • Note
  • Example

Syntax

T = n2T(n)

Inputs

Variable Symbol Description Format Units
n mean motion 1×1
double		 rad/s

Outputs

Variable Symbol Description Format Units
T orbital period 1×1
double		 s

Note

  • This function is only valid for circular () and elliptical () orbits.

Example

Find the period of an orbit around the Earth with mean motion .

Finding the orbital period in seconds,

T = n2T(0.0011);

Converting the orbital period to hours,

T = T/3600

T =

   1.586662956358481
