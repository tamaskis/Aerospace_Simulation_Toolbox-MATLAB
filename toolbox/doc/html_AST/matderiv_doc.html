matderiv

Syntax

dRdt = matderiv(R_A2B,omega_A2B_B)

Inputs

Variable Symbol Description Format Units

double		 rad/s

Outputs

Variable Symbol Description Format Units
dRdt time derivative of the passive rotation matrix from frame to frame 3×3
double		 1/s
