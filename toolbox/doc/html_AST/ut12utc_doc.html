ut12utc

UTC from UT1.

Syntax

MJD_UTC = ut12utc(MJD_UT1,DUT1)

Description

MJD_UTC = ut12utc(MJD_UT1,DUT1) determines UTC given UT1 and .

Inputs

Variable Symbol Description Format Units
MJD_UT1 UT1 (Universal Time 1) 1×1
double		 MJD
DUT1 difference between UT1 and UTC 1×1
double		 s

Outputs

Variable Symbol Description Format Units
MJD_UTC UTC (Universal Coordinated Time) 1×1
double		 MJD

Example

Find UTC corresponding to January 1, 2000, 12:00:00 UT1.

First, let's find the modified Julian date of UT1.

MJD_UT1 = cal2mjd([2000,1,1,12,0,0]);

Next, let's find the difference between UTC and UT1.

DUT1 = get_dut1(MJD_UT1);

Finding the modified Julian date of UTC,

MJD_UTC = ut12utc(MJD_UT1,DUT1);

Expressing UTC as a calendar date + hours, minutes, and seconds,

cal_UTC = mjd2cal(MJD_UTC);
year = cal_UTC(1)
month = cal_UTC(2)
day = cal_UTC(3)
hour = cal_UTC(4)
minute = cal_UTC(5)
second = cal_UTC(6)
