cal2mjd

Modified Julian date from Gregorian (calendar) date.

Syntax

MJD = cal2mjd(cal)
MJD = cal2mjd([YYYY,MM,DD,hh,mm,ss])

Description

MJD = cal2mjd(cal) converts a Gregorian date to a modified Julian date.

MJD = cal2mjd([YYYY,MM,DD,hh,mm,ss]) does the same as the syntax above (this syntax is meant to demonstrate the format of the input cal).

Input/Output Parameters

Variable Symbol Description Format Units
Input cal Gregorian (calendar) date (input as [YYYY,MM,DD,hh,mm,ss]) 1×6
double		 [y,mo,d,h,m,s]
Output MJD modified Julian date 1×1
double		 d

Example

Find the modified Julian date of January 1, 2000, 12:00:00.

MJD = cal2mjd([2000,1,1,12,0,0])

MJD =

   5.1544e+04
