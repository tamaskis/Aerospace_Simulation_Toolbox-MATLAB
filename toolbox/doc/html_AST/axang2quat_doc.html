axang2quat

Axis-angle representation to unit quaternion.

Syntax

q = axang2quat(e,Phi)

Description

q = axang2quat(e,Phi) finds the unit quaternion corresponding to the specified principal rotation vector and principal angle.

Inputs

Variable Symbol Description Format Units
e principal rotation vector 3×1
double		 -
Phi principal angle 1×1
double		 rad

Outputs

Variable Symbol Description Format Units
q unit quaternion 4×1
double		 -

Note

  • does not have to be input as a unit vector.
  • The unit quaternion, , is returned using the scalar-first convention, where is the scalar component and is the vector component.
  • The scalar component, , is chosen to be positive.

Example

Find the unit quaternion corresponding to a rotation of about the axis .

Defining the principal rotation vector,

e = (sqrt(3)/3)*[1;1;1];

Defining the principal angle,

Phi = pi/3;

Computing the unit quaternion,

q = axang2quat(e,Phi)

q =

    0.8660
    0.2887
    0.2887
    0.2887
