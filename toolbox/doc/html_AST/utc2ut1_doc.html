utc2ut1

UT1 from UTC.

Syntax

MJD_UT1 = utc2ut1(MJD_UTC,DUT1)

Description

MJD_UT1 = utc2ut1(MJD_UTC,DUT1) determines UT1 given UTC and .

Inputs

Variable Symbol Description Format Units
MJD_UTC UTC (Universal Coordinated Time) 1×1
double		 MJD
DUT1 difference between UT1 and UTC 1×1
double		 s

Outputs

Variable Symbol Description Format Units
MJD_UT1 UT1 (Universal Time 1) 1×1
double		 MJD

Example

Find UT1 corresponding to January 1, 2000, 12:00:00 UTC.

First, let's find the modified Julian date of UTC.

MJD_UTC = cal2mjd([2000,1,1,12,0,0]);

Next, let's find the difference between UT1 and UTC.

DUT1 = get_dut1(MJD_UTC);

Finding the modified Julian date of UT1,

MJD_UT1 = utc2ut1(MJD_UTC,DUT1);

Expressing UT1 as a calendar date + hours, minutes, and seconds,

cal_UT1 = mjd2cal(MJD_UT1);
year = cal_UT1(1)
month = cal_UT1(2)
day = cal_UT1(3)
hour = cal_UT1(4)
minute = cal_UT1(5)
second = cal_UT1(6)
