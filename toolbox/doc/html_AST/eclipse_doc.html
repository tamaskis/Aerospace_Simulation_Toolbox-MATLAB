eclipse

Determines if a satellite is in eclipse (cylindrical shadow model).

Syntax

in_eclipse = eclipse(r,r_sun)

Description

in_eclipse = eclipse(r,r_sun) determines if a satellite is in eclipse given the position of the satellite, and the position of the Sun, (resolved in the same coordinate frame).

Input/Output Parameters

Variable Symbol Description Format Units
Input r satellite position 3×1
double		 m
r_sun Sun position 3×1
double		 m
Output in_eclipse - true if satellite is in eclipse, false otherwise 1×1
logical		 -

Note

and must be resolved in the same coordinate frame.
