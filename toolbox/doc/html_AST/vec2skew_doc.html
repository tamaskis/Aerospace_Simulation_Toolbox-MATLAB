vec2skew

Converts a vector to a skew-symmetric matrix representing a cross product.

Syntax

ax = vec2skew(a)

Description

ax = vec2skew(a) finds the skew-symmetrix matrix that can be used to compute the cross product between two arbitary vectors and as .

Inputs

Variable Symbol Description Format Units
a vector 3×1
double		 -

Outputs

Variable Symbol Description Format Units
ax skew-symmetrix matrix where 3×3
double		 -

Example

Consider the vectors and .

a = [1;2;3];
b = [-4;2;-11];

We can find the cross product as

cross(a,b)

ans =

   -28
    -1
    10

Alternatively, we can first find the cross product matrix .

ax = vec2skew(a)

ax =

     0    -3     2
     3     0    -1
    -2     1     0

Multiplying by will yield the same result as performing directly.

ax*b

ans =

   -28
    -1
    10
