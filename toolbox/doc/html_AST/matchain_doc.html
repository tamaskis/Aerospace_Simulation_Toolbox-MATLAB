matchain

Chaining passive rotations represented by rotation matrices.

Syntax

R_A2C = matchain(R_A2B,R_B2C)

Description

R_A2C = matchain(R_A2B,R_B2C) returns the rotation matrix representing the rotation from frame to frame , given the rotation matrices representing the rotations from frame to frame and then from frame to frame ( and , respectively).

Inputs

Variable Symbol Description Format Units
R_A2B rotation matrix representing rotation from frame to frame 4×1
double		 -
R_B2C rotation matrix representing rotation from frame to frame 3×3
double		 -

Outputs

Variable Symbol Description Format Units
R_A2C rotation matrix representing rotation from frame to frame 3×3
double		 -

Example

First, let's construct two rotation matrices representing rotations from frame to frame and then from frame to frame .

R_A2B = eul2mat_321(pi/2,pi/4,-pi/3);
R_B2C = eul2mat_321(-pi/6,0,pi/5);

The rotation matrix from frame to frame is then

R_A2C = matchain(R_A2B,R_B2C)

R_A2C =

    0.2500    0.9186   -0.3062
   -0.8594    0.0648   -0.5073
   -0.4461    0.3899    0.8056
