Syntax

theta_deg = rad2deg(theta_rad)

Description

theta_deg = rad2deg(theta_rad) converts an angle from radians () to degrees ().

Input/Output Parameters

Variable Symbol Description Format Units
Input theta_rad angle in radians 1×1
double		 rad
Output theta_deg angle in degrees 1×1
double		 °

Example

Convert to degrees.

theta_rad = 1;
theta_deg = rad2deg(theta_rad)

theta_deg =

   57.2958
