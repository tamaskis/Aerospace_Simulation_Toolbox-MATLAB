Syntax

[e,Phi] = eul2axang_321(psi,theta,phi)

Description

[e,Phi] = eul2axang_321(psi,theta,phi) finds the principal rotation vector and principal angle corresponding to the specified 3-2-1 Euler angles (yaw, pitch, and roll).

Inputs

Variable Symbol Description Format Units
psi yaw angle (1st rotation, about 3rd axis) 1×1
double		 rad
theta pitch angle (2nd rotation, about 2nd axis) 1×1
double		 rad
phi roll angle (3rd rotation, about 1st axis) 1×1
double		 rad

Outputs

Variable Symbol Description Format Units
e principal rotation vector 3×1
double		 -
Phi principal angle 1×1
double		 rad
