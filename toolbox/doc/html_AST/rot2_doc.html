rot2

Rotation matrix for a passive rotation about the 2nd axis.

Syntax

R2 = rot2(theta)

Description

R2 = rot2(theta) finds the rotation matrix defining the rotation matrix for a passive rotation of about the 2nd axis.

Inputs

Variable Symbol Description Format Units
theta angle of rotation 1×1
double		 rad

Outputs

Variable Symbol Description Format Units
R2 rotation matrix about 2nd axis (passive) 3×3
double		 -

Example

Find the rotation matrix for a rotation of about the 2nd axis.

Defining the angle in radians,

theta = 45*(pi/180);

Finding the rotation matrix ,

R2 = rot2(theta)

R2 =

    0.7071         0   -0.7071
         0    1.0000         0
    0.7071         0    0.7071
