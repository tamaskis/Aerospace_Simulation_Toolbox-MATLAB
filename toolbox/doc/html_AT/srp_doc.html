Syntax

f_srp = srp(r,r_sun,CR,Asrp,m)

Description

f_srp = srp(r,r_sun,CR,Asrp,m) determines the perturbing acceleration due to solar radiation pressure () given the satellite position (), Sun position (), coefficient of reflectivity (), SRP reference area (), and satellite mass ().
