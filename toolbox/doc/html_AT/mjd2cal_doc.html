mjd2cal

Gregorian (calendar) date from modified Julian date.

Back to Astrodynamics Toolbox Contents.

Syntax

cal = mjd2cal(MJD)

Description

cal = mjd2cal(MJD) converts a modified Julian date to a Gregorian date.

Input/Output Parameters

Variable Symbol Description Format Units
Input MJD modified Julian date 1×1
double		 d
Output cal Gregorian (calendar) date (output as [YYYY,MM,DD,hh,mm,ss]) 1×6
double		 [y,mo,d,h,m,s]

Example

Find the Gregorian date of 51544.5 MJD.

cal = mjd2cal(51544.5)

cal =

        2000           1           1          12           0           0
