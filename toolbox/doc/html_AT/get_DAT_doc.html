Syntax

DAT = get_DAT(MJD)

Description

DAT = get_DAT(MJD) returns the difference between TAI and UTC ().

Input/Output Parameters

Variable Symbol Description Format Units
Input MJD modified Julian date (any time scale) 1×1
double		 MJD
Output DAT difference between TAI and UTC 1×1
double		 s

Example

Obtain for January 1, 2000.

First, let's convert this calendar date to a modified Julian date.

MJD = cal2mjd([2000,1,1,0,0,0]);

Finding ,

DAT = get_DAT(MJD)
