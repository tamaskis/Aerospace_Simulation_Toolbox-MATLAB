Syntax

[C,S] = gravity_coeffs(C_norm,S_norm)

Description

[C,S] = gravity_coeffs(C_norm,S_norm) returns the functions C(n,m) and S(n,m) that return the un-normalized gravitational coefficients of arbitrary degree and order , and , given the arrays C_norm and S_norm storing the normalized gravitational coefficients, (C_norm) and (S_norm).
