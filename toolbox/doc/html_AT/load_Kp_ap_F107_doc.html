load_Kp_ap_F107

Loads space weather data.

Back to Astrodynamics Toolbox Contents.

Syntax

Kp_ap_F107 = load_Kp_ap_F107
Kp_ap_F107 = load_Kp_ap_F107(MJD_UTC0,duration)

Description

Kp_ap_F107 = load_Kp_ap_F107 loads the full space weather data set.

Kp_ap_F107 = load_Kp_ap_F107(MJD_UTC0,duration) loads the space weather data from 3 days before the start of the simulation until the end of the simulation. The start date is specified as the modified Julian date of the initial UTC, and the end of the simulation is calculated using the simulation duration (duration, specified in hours).
