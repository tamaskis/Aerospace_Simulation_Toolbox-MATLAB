utc2tai

TAI from UTC.

Syntax

MJD_TAI = utc2tai(MJD_UTC,DAT)

Description

MJD_TAI = utc2tai(MJD_UTC,DAT) determines TAI given UTC and .

Input/Output Parameters

Variable Symbol Description Format Units
Input MJD_UTC UTC (Universal Coordinated Time) 1×1
double		 MJD
DAT difference between TAI and UTC 1×1
double		 s
Output MJD_TAI TAI (International Atomic Time) 1×1
double		 MJD

Example

Find TAI corresponding to January 1, 2000, 12:00:00 UTC.

First, let's find the modified Julian date of UTC.

MJD_UTC = cal2mjd([2000,1,1,12,0,0]);

Next, let's find the difference between TAI and UTC.

DAT = get_DAT(MJD_UTC);

Finding the modified Julian date of TAI,

MJD_TAI = utc2tai(MJD_UTC,DAT);

Expressing TAI as a calendar date + hours, minutes, and seconds,

cal_TAI = mjd2cal(MJD_TAI);
year = cal_TAI(1)
month = cal_TAI(2)
day = cal_TAI(3)
hour = cal_TAI(4)
minute = cal_TAI(5)
second = cal_TAI(6)
