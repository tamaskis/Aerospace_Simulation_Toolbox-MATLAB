eci2ecef_matrix_approx

Rotation matrix from Earth-centered inertial (ECI) frame to Earth-centered Earth-fixed (ECEF) frame (approximate).

Syntax

R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1)
R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'Linear')
R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'IAU2006/2000')
R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'Vallado')

Description

R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1) finds the rotation matrix from the Earth-centered inertial (ECI) frame to the Earth-Centered Earth-fixed (ECEF) frame, , given the modified Julian date of UT1, . This syntax uses the gmst_linear function to calculate the Greenwich mean sidereal time (GMST), .

R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'Linear') does the same as the syntax above.

R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'IAU2006/2000') finds the same rotation matrix, but uses the gmst_iau06 function to calculate the GMST.

R_eci2ecef = eci2ecef_matrix_approx(MJD_UT1,'Vallado') finds the same rotation matrix, but uses the gmst_vallado function to calculate the GMST.
