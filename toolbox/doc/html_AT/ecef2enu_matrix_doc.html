Syntax

R_ecef2enu = ecef2enu(r0_ecef)

Description

R_ecef2enu = ecef2enu(r0_ecef) finds the rotation matrix from the Earth-centered Earth-fixed (ECEF) frame to the East, North, Up (ENU) frame, , given the position of the ENU frame's origin resolved in the ECEF frame, .

Input/Output Parameters

Variable Symbol Description Format Units
Input r0_ecef origin of ENU frame resolved in ECEF frame 3×1
double		 any
Output R_ecef2enu rotation matrix from ECEF frame to ENU frame 3×3
