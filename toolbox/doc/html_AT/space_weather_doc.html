space_weather

Space weather data for atmospheric models.

Syntax

sw = space_weather(MJD_UT1,sw1,sw2,model)

Description

sw = space_weather(MJD_UT1,sw1,sw2,model) returns a structure sw (defined below) storing the space weather parameters needed by various atmospheric models, given the modified Julian date of UT1 (), the arrays sw1 and sw2 storing space weather data, and the atmospheric model being used.

Output sw - space weather data
