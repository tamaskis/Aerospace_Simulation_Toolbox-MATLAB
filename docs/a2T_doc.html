a2T

Orbital period from semi-major axis.

Syntax

T = a2T(a)
T = a2T(a,mu)

Description

T = a2T(a) finds the orbital period, , given the semi-major axis, , for a circular () or elliptical () orbit around the Earth.

T = a2T(a,mu) finds the orbital period, , given the semi-major axis, , for a circular () or elliptical () orbit around a cental body with gravitational parameter .

Input/Output Parameters

Variable Symbol Description Format Units
Input a semi-major axis 1×1
double		 m
mu (OPTIONAL) gravitational parameter 1×1
double		 m3/s2
Output T orbital period 1×1
double		 s

Example #1: Orbit around the Earth.

Find the period of an orbit around the Earth with semi-major axis .

Defining the semi-major axis in meters,

a = 7000*1000;

Finding the orbital period (in ),

T = a2T(a);

Converting the orbital period to hours,

T = T/3600

T =

    1.6190

Example #2: Orbit around the moon.

Find the period of an orbit around the Moon with semi-major axis .

Defining the semi-major axis in meters,

a = 3000*1000;

Gravitational parameter of the moon (in ),

mu = MU_MOON;

Finding the orbital period (in ),

T = a2T(a,mu)

T =

   1.4745e+04

Converting the orbital period to hours,

T = T/3600

T =

    4.0958
