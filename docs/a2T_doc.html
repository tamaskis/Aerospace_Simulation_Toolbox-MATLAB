a2T

Orbital period from semi-major axis.

Syntax

T = a2T(a,mu)

Inputs

Variable Symbol Description Format Units
a semi-major axis 1×1
double		 m
mu standard gravitational parameter 1×1
double		 m3/s2

Outputs

Variable Symbol Description Format Units
T orbital period 1×1
double		 s

Note

  • This function is only valid for circular () and elliptical () orbits.

Example #1: Orbit around the Earth.

Find the period of an orbit around the Earth with semi-major axis .

Defining the semi-major axis in meters,

a = 7000*1000;

Standard gravitational parameter of the Earth (in ),

mu = MU_EARTH;

Finding the orbital period (in ),

T = a2T(a,mu);

Converting the orbital period to hours,

T = T/3600

T =

   1.619032399966496

Example #2: Orbit around the Moon.

Find the period of an orbit around the Moon with semi-major axis .

Defining the semi-major axis in meters,

a = 3000*1000;

Standard gravitational parameter of the Moon (in ),

mu = MU_MOON;

Finding the orbital period (in ),

T = a2T(a,mu)

T =

     1.474482615552693e+04

Converting the orbital period to hours,

T = T/3600

T =

   4.095785043201925
