rot313

Rotation matrix for the 3-1-3 rotation sequence.

Syntax

R313 = rot313(theta1,theta2,theta3)

Description

R313 = rot313(theta1,theta2,theta3) finds the rotation matrix for the 3-1-3 rotation sequence, which is defined as follows:

  1. 1st rotation = rotation of about the 3rd axis.
  2. 2nd rotation = rotation of about the 1st axis.
  3. 3rd rotation = rotation of about the 3rd axis.

Inputs

Variable Symbol Description Format Units
theta1 angle for 1st rotation (about 3rd axis) 1×1
double		 rad
theta2 angle for 2nd rotation (about 1st axis) 1×1
double		 rad
theta3 angle for 3rd rotation (about 3rd axis) 1×1
double		 rad

Outputs

Variable Symbol Description Format Units
R313 rotation matrix for 3-1-3 rotation sequence 3×3
double		 -

Example

Find the rotation matrix for the following 3-1-3 rotation sequence:

  1. _Rotation of about the 3rd axis.
  2. _Rotation of about the 1st axis.
  3. _Rotation of about the 3rd axis.

Defining the angles in radians,

theta1 = 30*(pi/180);
theta2 = 40*(pi/180);
theta3 = 50*(pi/180);

Finding the rotation matrix ,

R313 = rot313(theta1,theta2,theta3)

R313 =

    0.2633    0.8296    0.4924
   -0.9096    0.0434    0.4132
    0.3214   -0.5567    0.7660
