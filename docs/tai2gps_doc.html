tai2gps

GPS time from TAI.

Syntax

MJD_GPS = tai2gps(MJD_TAI)

Description

MJD_GPS = tai2gps(MJD_TAI) determines GPS time given TAI.

Input/Output Parameters

Variable Symbol Description Format Units
Input MJD_TAI TAI (International Atomic Time) 1×1
double		 MJD
Output MJD_GPS GPS time 1×1
double		 MJD

Example

Find the GPS time corresponding to January 1, 2000, 12:00:00 TAI.

First, let's find the modified Julian date of TAI.

MJD_TAI = cal2mjd([2000,1,1,12,0,0]);

Finding the corresponding modified Julian date of GPS time,

MJD_GPS = tai2gps(MJD_TAI);

Expressing the GPS time as a calendar date + hours, minutes, and seconds,

cal_GPS = mjd2cal(MJD_GPS);
year = cal_GPS(1)
month = cal_GPS(2)
day = cal_GPS(3)
hour = cal_GPS(4)
minute = cal_GPS(5)
second = cal_GPS(6)

year =

        2000


month =

     1


day =

     1


hour =

    11


minute =

    59


second =

   41.0000
