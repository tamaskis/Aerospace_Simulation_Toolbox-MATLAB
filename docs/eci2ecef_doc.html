Syntax

[r_ecef,v_ecef] = eci2ecef(r_eci,v_eci,w_eci,R_eci2ecef)

Description

[r_ecef,v_ecef] = eci2ecef(r_eci,v_eci,w_eci,R_eci2ecef) determines the ECEF position and velocity ( and ) given the ECI position and velocity ( and ), the angular velocity of the Earth (), and the rotation matrix from the ECI frame to the ECEF frame ().

Input/Output Parameters

Variable Symbol Description Format Units
Input r_eci position resolved in ECI frame 3×1
double		 any
v_eci inertial velocity resolved in ECI frame 3×1
double		 any
w_eci Earth angular velocity resolved in ECI frame 3×1
double		 rad/s
R_eci2ecef rotation matrix from ECI frame to ECEF frame 3×3
double		 -
Output r_ecef position resolved in ECEF frame 3×1
double		 same as r_eci
v_ecef ECEF velocity resolved in ECEF frame 3×1
double		 same as v_eci

Note

  • and can be input in any units, but they must be consistent. The units of will match those of , and the units of will match those of .
  • By definition, the angular velocity of the Earth is defined as the angular velocity of the ECEF frame with respect to the ECI frame.
