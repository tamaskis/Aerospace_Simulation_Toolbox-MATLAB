fund_arg_iau06

Fundamental arguments of nutation theory (IAU2006/2000, CIO based).

Back to Astrodynamics Toolbox Contents.

Syntax

F = fund_arg_iau06(T_TT)

Description

F = fund_arg_iau06(T_TT) determines the fundamental arguments of nutation theory given the Julian centuries since J2000.0 of TT (Terrestrial Time).

Input/Output Parameters

Variable Symbol Description Format Units
Input T_TT Julian centuries since J2000.0 of TT (Terrestrial Time) 1×1
double		 c
Output F fundamental arguments of nutation theory 14×1
double		 rad
