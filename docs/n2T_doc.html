n2T

Syntax

T = n2T(n)

Description

T = n2T(n) finds the orbital period, , given the mean motion, , for a circular () or elliptical () orbit.

Input/Output Parameters

Variable Symbol Description Format Units
Input n mean motion 1×1
double		 rad/s
Output T orbital period 1×1
double		 s

Example

Find the period of an orbit around the Earth with mean motion .

Finding the orbital period in seconds,

T = n2T(0.0011);

Converting the orbital period to hours,

T = T/3600

T =

    1.5867
