quatrotate

Passive rotation of a vector by a unit quaternion.

Syntax

rB = quatrotate(q_A2B,rA)

Description

rB = quatrotate(q_A2B,rA) takes a vector, , expressed in frame , and finds its coordinates in frame , given the unit quaternion representing the rotation from frame to frame .

Inputs

Variable Symbol Description Format Units
q_A2B unit quaternion representing rotation from frame to frame 4×1
double		 -
rA vector expressed in frame 3×1
double		 -

Outputs

Variable Symbol Description Format Units
rB vector expressed in frame 3×1
double		 -

Note

  • This function assumes the scalar-first convention for quaternions.
  • This function normalizes the input quaternion.

Example

First, let's construct a quaternion representing the rotation from frame to frame from a set of 3-2-1 Euler angles.

q_A2B = eul2quat_321(pi/2,pi/4,-pi/3);

Next, let's define the coordinate vector representing a vector expressed in frame .

rA = [5;4;3];

Expressing the vector in frame ,

rB = quatrotate(q_A2B,rA)

rB =

    0.7071
   -6.7866
   -1.8553
