quat2axang

Quaternion to axis-angle representation.

Syntax

[e,Phi] = quat2axang(q)

Description

[e,Phi] = quat2axang(q) finds the principal rotation vector and principal angle corresponding to the specified quaternion.

Inputs

Variable Symbol Description Format Units
q quaternion 4×1
double		 -

Outputs

Variable Symbol Description Format Units
e principal rotation vector 3×1
double		 -
Phi principal angle 1×1
double		 rad

Note

  • The quaternion, , should be input using the scalar-first convention, where is the scalar component and is the vector component.
  • does not have to be input as a unit quaternion.
  • If , then is returned as .

Example

Find the axis-angle representation of the rotation specified by the unit quaternion

Defining the unit quaternion,

q = [0.3827;0.1562;0.7808;-0.4685];

Computing the axis-angle representation,

[e,Phi] = quat2axang(q)

e =

    0.1691
    0.8451
   -0.5071


Phi =

    2.3562
