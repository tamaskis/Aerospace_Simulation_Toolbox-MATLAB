quatslerp

Syntax

q = quatslerp(q1,q2,t)

Description

q = quatslerp(q1,q2,t) interpolates a quaternion that is at the proportion along the great arc between and on the unit sphere.

Inputs

q1 unit quaternion 4×1
double		 -

double		 -
t interpolation parameter 1×1
double		 -

Outputs

Variable Symbol Description Format Units
q interpolated unit quaternions 4×1
double		 -

Note

  • This function assumes the scalar-first convention for quaternions.
  • This function assumes that and are input as unit quaternions, so it does not normalize them.
  • is normalized.
  • The scalar part of is chosen to be positive.

Example #2

Let's define two quaternions we want to interpolate between via SLERP.

q1 = [0.9173;0.3023;0.0655;0.2508];
q2 = [0.1826;-0.3651;-0.5477;-0.7303];

Performing SLERP at a large number of points between and ,

t = 0:0.01:1;
q = zeros(4,length(t));
for i = 1:length(t)
    q(:,i) = quatslerp(q1,q2,t(i));
end

Let's rotate the unit vector through each of the orientations we just defined to trace out a path on the unit sphere. Note that we use the conjugate of since we are performing an active rotation in this case, and quatrotate expects a quaternion representing a passive rotation.

e = [1;0;0];
e_rotated = zeros(3,length(t));
for i = 1:length(t)
    e_rotated(:,i) = quatrotate(quatconj(q(:,i)),e);
end

Plotting the path on the unit sphere,

[X,Y,Z] = sphere;
figure;
hold on;
surf(X,Y,Z,'FaceColor',[0.75,0.75,0.75])
plot3(e_rotated(1,:),e_rotated(2,:),e_rotated(3,:),'LineWidth',2.5,...
    'Color','black');
axis off;
axis equal;
view([170,20]);
