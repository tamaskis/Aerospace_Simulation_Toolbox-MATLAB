quatinv

Inverse of a quaternion.

Syntax

q_inv = quatinv(q)

Description

q_inv = quatinv(q) returns the inverse of the specified quaternion.

Inputs

Variable Symbol Description Format Units
q quaternion 4×1
double		 -

Outputs

Variable Symbol Description Format Units
q_inv quaternion inverse 4×1
double		 -

Note

  • This function assumes the scalar-first convention for quaternions.
  • does not have to be input as a unit quaternion (this function is for all quaternions, not just unit quaternions).
  • is not normalized.

Example

Find the inverse of

Defining the quaternion,

q = [1;2;3;4];

Computing the quaternion inverse,

q_inv = quatinv(q)

q_inv =

    0.0333
   -0.0667
   -0.1000
   -0.1333

Verifying that the (Hamilton) product of the quaternion and its inverse yields the unit quaternion,

q_prod = quatmul(q,q_inv)

q_prod =

    1.0000
         0
         0
   -0.0000
