load_eop

Loads data storing Earth orientation parameters (IAU2006/2000, CIO based).

Back to Astrodynamics Toolbox Contents.

  • Syntax

Syntax

eop = load_eop
eop = load_eop(MJD_UTC0,duration)

Description

eop = load_eop loads the full data set storing the Earth orientation parameters.

eop = load_eop(MJD_UTC0,duration) loads the Earth orientation parameters from the start date until the end of the simulation. The start date is specified as the modified Julian date of the initial UTC, and the end of the simulation is calculated using the simulation duration (duration, specified in hours).
