wks2gps

GPS time from GPS week and seconds.

Syntax

MJD_GPS = wks2gps(wk,s)

Description

MJD_GPS = wks2gps(wk,s) determines the GPS time given the GPS week and seconds.

Input/Output Parameters

Variable Symbol Description Format Units
Input wk GPS week 1×1
double		 wk
s GPS seconds 1×1
double		 s
Output MJD_GPS GPS time 1×1
double		 MJD

Example

Find the GPS time (as a Gregorian date) corresponding to GPS week 2195 and 250527 seconds.

First, let's find the modified Julian date of GPS time.

MJD_GPS = wks2gps(2195,250527);

Finding the corresponding Gregorian date,

cal = mjd2cal(MJD_GPS);
YYYY = cal(1)
MM = cal(2)
DD = cal(3)
hh = cal(4)
mm = cal(5)
ss = cal(6)

YYYY =

        2022


MM =

     2


DD =

     1


hh =

    21


mm =

    35


ss =

   27.0000
