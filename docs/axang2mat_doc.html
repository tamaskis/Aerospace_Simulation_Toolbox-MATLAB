axang2mat

Axis-angle representation to rotation matrix.

Syntax

R = axang2mat(e,Phi)

Description

R = axang2mat(e,Phi) finds the rotation matrix corresponding to the specified principal rotation vector and principal angle.

Inputs

Variable Symbol Description Format Units
e principal rotation vector 3×1
double		 -
Phi principal angle 1×1
double		 rad

Outputs

Variable Symbol Description Format Units
R rotation matrix 3×3
double		 -

Note

  • This function normalizes .

Example

Find the rotation matrix corresponding to a rotation of about the axis .

Defining the principal rotation vector,

e = (sqrt(3)/3)*[1;1;1];

Defining the principal angle,

Phi = pi/3;

Computing the rotation matrix,

R = axang2mat(e,Phi)

R =

    0.6667    0.6667   -0.3333
   -0.3333    0.6667    0.6667
    0.6667   -0.3333    0.6667
