Syntax

n = T2n(T)

Description

n = T2n(T) finds the mean motion, , given the orbital period, , for a circular () or elliptical () orbit.

Input/Output Parameters

Variable Symbol Description Format Units
Input T orbital period 1×1
double		 s
Output n mean motion 1×1
double		 rad/s

Example

Find the mean motion for an orbit around the Earth with orbital period .

Defining the orbital period in seconds,

T = 1.5*3600;

Finding the mean motion (),

n = T2n(T)

n =

    0.0012
