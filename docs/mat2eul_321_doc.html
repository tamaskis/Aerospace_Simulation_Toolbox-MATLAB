mat2eul_321

Rotation matrix to 3-2-1 Euler angles (yaw, pitch, and roll).

Syntax

[psi,theta,phi] = mat2eul_321(R)

Description

[psi,theta,phi] = mat2eul_321( R ) finds the 3-2-1 Euler angles (yaw, pitch, and roll) corresponding to the specified rotation matrix.

Inputs

Variable Symbol Description Format Units
R rotation matrix 3×3
double		 -

Outputs

Variable Symbol Description Format Units
psi yaw angle (1st rotation, about 3rd axis) 1×1
double		 rad
theta pitch angle (2nd rotation, about 2nd axis) 1×1
double		 rad
phi roll angle (3rd rotation, about 1st axis) 1×1
double		 rad

Note

  • When a singularity corresponding to the pitch angles of is encountered, the yaw angle () is set to 0 and the roll angle () is determined accordingly.

Example

Find the 3-2-1 Euler angles corresponding to the rotation matrix

Defining the rotation matrix,

R = [ 2/3   2/3  -1/3;
     -1/3   2/3   2/3;
      2/3  -1/3   2/3];

Computing the 3-2-1 Euler angles,

[psi,theta,phi] = mat2eul_321(R)

psi =

    0.7854


theta =

    0.3398


phi =

    0.7854
