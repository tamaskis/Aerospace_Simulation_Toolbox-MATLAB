Syntax

r = quatmul(p,q)

Description

r = quatmul(p,q) returns the Hamilton product of p and q.

Inputs

double		 -

Note

  • This function assumes the scalar-first convention for quaternions.
  • and do not have to be input as unit quaternions, and is not returned as a unit quaternion.
