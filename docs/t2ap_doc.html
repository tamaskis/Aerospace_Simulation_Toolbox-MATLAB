t2ap

Syntax

ap = t2ap(MJD_UT1,MJD_UT1_data,ap_data)

Description

ap = t2ap(MJD_UT1,MJD_UT1_data,ap_data) returns the planetary amplitude, , corresponding to the specified time, , given the arrays MJD_UT1_data and ap_data that tabulate the planetary amplitude data as a function of time.

Input/Output Parameters

Variable Symbol Description Format Units
Input MJD_UT1 UT1 (Universal Time 1) 1×1
double		 MJD
MJD_UT1_data - UT1 for each data entry N×1
double		 MJD
ap_data - array storing planetary amplitude data
- Column 1: -- planetary amplitude (00:00-03:00)
- Column 2: -- planetary amplitude (03:00-06:00)
- Column 3: -- planetary amplitude (06:00-09:00)
- Column 4: -- planetary amplitude (09:00-12:00)
- Column 5: -- planetary amplitude (12:00-15:00)
- Column 6: -- planetary amplitude (15:00-18:00)
- Column 7: -- planetary amplitude (18:00-21:00)
- Column 8: -- planetary amplitude (21:00-00:00)		 N×8
double		 γ
Output ap planetary amplitude at beginning of 3-hour interval containing specified 1×1
double		 γ

Note

N = # of data points
