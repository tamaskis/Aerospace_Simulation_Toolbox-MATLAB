Syntax

theta_deg = arcsec2deg(theta_arcsec)

Description

theta_deg = arcsec2deg(theta_arcsec) converts an angle from arcseconds () to degrees ().

Input/Output Parameters

Variable Symbol Description Format Units
Input theta_arcsec angle in arcseconds 1×1
double		 ''
Output theta_deg angle in degrees 1×1
double		 °

Example

Convert to degrees.

theta_arcsec = 1;
theta_deg = arcsec2deg(theta_arcsec)

theta_deg =

   2.7778e-04
